## ✨ What is this?

This isn't just a website.

It's a 3D galaxy built from memories. Every star, every orbiting polaroid, every glowing planet — all of it exists for one person.

**Deanna** — my prettiest snake, my favorite human, my universe.

---

## 🌌 Features

```
🏠  Landing Page      — Romantic dark-space UI with aurora effects
💕  Reasons I Love You — 20 flip cards, each hiding a truth
📖  Our Story          — An animated timeline of us
💌  Love Notes         — Sticky notes with words from my heart  
🎁  Special Surprise   — A locked message, just for her
🌌  Galaxy Mode        — Full 3D interactive space experience
```

### Galaxy Includes:
- 🪐 **Ultra-realistic planet** — PBR shader, iridescence, subsurface scattering
- 🌠 **Shooting stars** — randomly crossing the sky
- 🌌 **Distant galaxies** — glowing in the background
- 🪨 **Asteroid belt** — orbiting the planet
- 🌙 **Moon** — with crater detail, orbiting in real-time
- 📸 **20 polaroid memories** — orbiting as cards in space
- 📹 **Video node** — a floating screen playing our memory
- ✨ **5,000+ stars** + particle belt + cosmic dust

```bash
git clone https://github.com/Fokiimay/Deanna-my-preetiest-snake.git
cd Deanna-my-preetiest-snake
npm install
npm run dev
```
